React Native / Expo / TypeScript

一个离线优先的移动端衣橱管理 App，用来记录衣物、整理分类标签、追踪穿着频率，并通过日历回看每天的穿搭。

项目概览

平台 Android / iOS

基于 Expo Dev Client 验证原生能力。

数据策略 Offline-first

衣橱、标签、穿搭和图片数据默认保存在本地。

角色 独立开发

覆盖产品设计、移动端实现、发布页面和隐私政策。

核心功能

衣物管理

添加、编辑和删除衣物，记录图片、名称、购买日期、穿着次数、上次穿着日期等信息。

分类与标签

支持自定义分类和标签，并按季节、场景、颜色或收纳状态快速筛选衣物。

穿搭日历

按日期保存每日穿搭照片，回看历史搭配，帮助减少重复穿搭和临时决策成本。

统计与备份

汇总衣物数量、分类分布和标签分布，并支持将数据与图片打包导出、恢复。

技术亮点

本地优先的数据模型

使用 AsyncStorage 管理衣物、分类、标签和穿搭记录，把核心使用路径设计成无需账号、无需网络也能运行。

图片与文件处理

集成 expo-image、expo-image-picker、expo-file-system、document picker 和 unzip 能力，完成图片选择、展示、备份导出与恢复。

跨页面状态与筛选体验

使用 Expo Router 和 React Navigation Tabs 组织衣橱、日历、个人页等核心页面，并维护筛选、排序和统计的业务状态。

发布配套页面

额外制作 GitHub Pages 展示页、隐私政策页面和 Google Play feature graphic，让项目具备可分享、可审核的完整外壳。

项目范围

  • 设计并实现衣物列表、详情编辑、批量添加、日历记录和个人统计页面。
  • 搭建分类、标签、排序、筛选、穿着次数和最近穿着时间等业务逻辑。
  • 处理图片选择、图片展示、背景移除入口、文件备份和恢复流程。
  • 整理 README、隐私政策、展示页面和应用发布所需的基础素材。
